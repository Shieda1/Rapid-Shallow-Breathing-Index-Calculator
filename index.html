The rapid shallow breathing index calculator is a tool that counts a parameter used for patients put on the mechanical ventilation.
How to calculate rapid shallow breathing index?
To calculate rapid shallow breathing index you should use the following formula:
rapid shallow breathing index = (respiratory rate / tidal volume) * 1000
